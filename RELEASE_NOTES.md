# Undiscord Python GUI v1.0.0 Release

Undiscord Python GUI의 첫 번째 공식 안정화 버전(v1.0.0) 릴리즈입니다.
이 도구는 기존의 웹 브라우저용 Userscript를 기반으로, 일반 사용자들이 보다 쉽고 안전하게 Discord 메시지를 일괄 검색하고 일방향/다방향 삭제 처리를 수행할 수 있도록 파이썬 Tkinter를 사용해 독자적인 다크 모드 GUI로 구축되었습니다.

---

## 🚀 주요 특징 (Key Features)

1. **Discord 스타일 다크 테마 GUI**
   - 가독성이 뛰어나고 미려한 디스코드 스타일의 테마 인터페이스를 탑재했습니다.
2. **토큰 암호화 자동 보호**
   - 민감한 디스코드 API 인증 토큰을 로컬에 평문으로 저장하지 않고, 사용자가 지정한 **마스터 비밀번호**로 강력하게 대칭 키 암호화(PBKDF2 + Fernet) 처리하여 보관합니다.
3. **한영키 및 Caps Lock 상태 실시간 감지**
   - 비밀번호 입력 시 오입력으로 인한 검증 에러를 예방하기 위해, 한영키(한글 모드) 및 Caps Lock 키 활성화 여부를 실시간으로 판별해 경고 안내를 제공합니다.
4. **시작 시 시각 자동 갱신**
   - 프로그램을 기동할 때 드롭다운 설정(예: 1일 전, 현재 시각 등)에 부합하게 삭제 범위 시간을 실행 시점 기준으로 자동 갱신합니다.
5. **네트워크 보안 레이어**
   - SSL/TLS 암호화 요청 강제, 만인간 공격(MitM)을 유도하는 로컬 프록시 우회 및 위조 도메인 접속 차단용 URL 화이트리스트 검사기 등을 빌트인으로 탑재하여 사용자의 통신 패킷 탈취를 철저히 예방합니다.

## 📦 배포 패키지 구성품
- **`UndiscordGUI.exe`**: 추가적인 설치 없이 즉시 단독 실행이 가능한 윈도우 무설치 포터블 버전입니다.
- **`UndiscordGUI_Release.zip`**: 실행파일 및 README, PATCH_NOTES 설명서가 결합된 통합 압축 배포용 패키지입니다.

---

## 🛠️ 실행 및 사용법 (How to Run)
1. 아래 첨부파일 중 `UndiscordGUI.exe` 또는 `UndiscordGUI_Release.zip`을 다운로드합니다.
2. 실행 시 토큰을 안전하게 암호화 보관하기 위해 마스터 비밀번호 설정(최소 4자) 팝업이 출력되므로, 비밀번호를 설정합니다. (추후 앱을 켤 때마다 본인 검증용으로 요구됩니다.)
3. 복사한 디스코드 인증 토큰, 삭제를 원하는 서버 ID 및 채널 ID를 입력하고 **"삭제 시작"** 버튼을 누릅니다.
